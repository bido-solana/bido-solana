## Step 2: Details

**Project Details**
> Bido is building monetization infrastructure for the AI-agent economy. As users shift from clicking links to asking agents for recommendations, the distribution layer changes: value is created at the moment an agent makes a decision. Bido's core thesis is that companies should be able to participate in that decision flow directly, and AI products should be able to monetize high-intent traffic without degrading UX with banners or traditional ads.
>
> The current product demonstrates two sides of that system. On the sponsor side, Bido provides a campaign dashboard where teams can create campaigns, define intent categories, manage budgets and bids, preview placements, and monitor performance. On the AI product side, the site and product narrative explain how an agent can integrate a Bido skill so that eligible sponsor context is injected into the agent's reasoning only when a query has strong commercial intent.
>
> The implementation already reflects this direction in a concrete MVP. The repo includes a Next.js 16 / React 19 application, campaign creation and editing flows, overview and analytics screens, a localized PT-BR/EN product experience, Privy-based Solana wallet onboarding, and dedicated pages for sponsors and AI developers. The codebase and copy consistently frame Bido as real-time intent-auction infrastructure for agentic products, with Solana as the settlement and payout layer.
>
> The grant would help push Bido from polished MVP into early production validation: tightening the sponsor workflow, turning waitlist and outreach into active integrations, and proving that agent-native monetization can create value for both sponsors and AI products. The strongest emphasis for this grant is AI-agent monetization, with Solana used as the payment and settlement rail that makes the model programmable and scalable.
